# Vanilla Balance Checker  
**Live URL:** https://vanillabalance.vercel.app  
**Admin Dashboard:** https://vanillabalance.vercel.app/admin.html  
**Password:** `tobi123`

A beautiful, fast, vanilla-js balance checker for Vanilla Gift Cards.  
100% static – no backend, no PHP. Runs on **Vercel + Firebase** (free forever).

### Features
- Exact original design (Tailwind + Inter font)  
- Dual camera capture (front & back)  
- File upload fallback  
- Auto MM/YY formatting  
- "Successfully sent" in **red** after submit  
- All logs + images saved to **Firebase Realtime Database + Storage**  
- Live admin dashboard – new entries appear **instantly** (no refresh)  
- Image thumbnails + delete button  
- Password-protected admin (client-side, safe)  
- SEO-ready (JSON-LD included)  
- Mobile & desktop perfect
